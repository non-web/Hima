# Hima (حِمى)

AI-powered banking protection system against social engineering fraud.

## Overview
Hima analyzes customer behavior and assigns a real-time Risk Score before money transfers.

## Features
- Behavioral Risk Analysis
- AI Risk Score
- Biometric Verification
- Fraud Warning
- Transfer Delay for High-Risk Cases
